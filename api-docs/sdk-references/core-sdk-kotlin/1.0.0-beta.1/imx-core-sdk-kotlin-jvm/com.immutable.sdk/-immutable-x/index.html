class ImmutableX(base: ImmutableXBase, nodeUrl: String?)

This is the entry point for the Immutable X SDK.

You can configure the environment or use any of the provided utility workflows which chain the necessary calls to perform standard actions (e.g. buy, sell etc).

Parameters

base

the environment the SDK will communicate with

nodeUrl

(optional) Ethereum node URL. This is only required for smart contract interactions.
