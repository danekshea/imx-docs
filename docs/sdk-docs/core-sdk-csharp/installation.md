# Installation

Add following nuget packages

https://www.nuget.org/packages/Imx.Sdk
https://www.nuget.org/packages/Imx.Sdk.Gen

```sh
dotnet add package Imx.Sdk --version 0.1.0
dotnet add package Imx.Sdk.Gen --version 0.1.0
```
